Execution history toolbar — fitting Current commit + sort without ugly wraps

The side panel can be as narrow as 220px or as wide as 500px+. Three design options, each specimen-tested at 240 / 320 / 440px. Pick one (I lean on A).

OPTION A

"Current commit" becomes a pill on the same row as the filters

Recommended

Move the checkbox into the filter pill row. It shares wrap behavior with the status filters, so the layout degrades gracefully: pills reflow onto the next line at narrow widths, and the sort icon sits on a tiny 24px trailing stub that never pushes anything. The commit pill is the checkbox — no second label row, no second toolbar tier.

Implementation delta from current

  • Drop the .hist-check block from its own row; give it the .pill class plus a .commit-pill modifier.
  • Keep the <input type="checkbox"> visually hidden inside the pill for a11y; the label is the button face.
  • Sort button gets its own tiny .iconbtn (24×24), same row — it never wraps because flex-wrap prefers wrapping whole pills first.
  • Toolbar becomes flex-direction: row; flex-wrap: wrap — remove the outer column.
OPTION B

Segmented control — scope first, sort as a compact trailing control

Alternative

Promote "All commits ↔ Current commit" to a first-class binary scope using a segmented control on its own row, with the sort dropdown pinned to the right. More explicit than a checkbox (shows both states at once), but costs you one full row. Best if you plan to add a third scope later (e.g. "My runs").

TL;DR

Pick one

  • Option A — smallest vertical footprint, most consistent with the existing pill vocabulary. The commit filter lives in the same mental bucket as the status filters ("what subset of runs do I want?"). Sort as a small icon-only button is fine because it's a rarely-changed preference. Best default.
  • Option B — use if you think "current commit" is a core mental mode worth promoting, or if you'll add more scopes later.
  • Option C — use if the panel will frequently be <240px and you want the filter row to dominate. Accepts a discoverability hit.
